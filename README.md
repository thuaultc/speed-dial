# :phone: Speed Dial

Hugo-based service for my own Speed Dial. Best used with the New Tab Redirect extension on Chrome.

## Link

See it [here](https://speed-dial.thuault.com).

## Build

### Build image

```bash
./build/build.sh
```

### Deploy on Kubernetes

```bash
./build/deploy.sh
```

### Update deployment image on Kubernetes

```bash
./build/update.sh
```
